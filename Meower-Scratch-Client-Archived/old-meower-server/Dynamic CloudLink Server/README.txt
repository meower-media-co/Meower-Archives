A. UPDATE YOUR COPY OF CLOUDLINK AND DEPENDENCIES
WIN: py -m pip install cloudlink pygithub pyngrok
UNIX/LINUX: python3 -m pip install cloudlink pygithub pyngrok

B. THEN, DO THE FOLLOWING:
1. Create a repo named "serverbots".
2. Turn on GitHub pages for your repo.
